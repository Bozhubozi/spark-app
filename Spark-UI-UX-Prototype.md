# Spark UI/UX 原型文档 v2.0

> 交互设计稿 | 2026-06-16 | 暗色主题 · 移动端 · Soul 风格做减法

---

## 1. 信息架构

```
App 入口
├── 启动屏（Splash，Logo + Slogan，1.5s）
├── 发现页（无需注册，可刷 5 张预览卡片）
│   └── 右滑/发消息/查看星座适配 → 触发注册
├── 注册拦截弹窗（微信一键登录 + 手机号 + 实名）
├── 注册后引导
│   ├── 兴趣标签（8 类 × 80+ 标签，至少 5 个，必做）
│   ├── 人格测试（10 题，可跳过，完成后解锁 AI 人格报告）
│   └── 虚拟形象（6-8 预设风格，可跳过，默认分配）
└── 主页（底部 3 Tab）
    ├── Tab 1: 发现——卡片滑动（标签+星座+人格数据为主）
    │   ├── 顶部筛选栏：星座 / 兴趣 / 同城
    │   ├── 匹配弹窗——全屏，含 AI 星座适配报告
    │   └── 每日运势入口（卡片顶部小横幅）
    ├── Tab 2: 聊天——聊天室列表 + 聊天详情
    │   ├── AI 破冰话题（匹配后自动推送）
    │   └── 长按消息：拉黑/举报
    └── Tab 3: 我的
        ├── 人格卡片（可分享）
        ├── 星座运势（每日更新）
        ├── 我的兴趣标签（可编辑）
        ├── 虚拟形象（可更换预设）
        └── 设置/登出
```

## 3. Design System Tokens（不变）

| Token | 值 | 用途 |
|---|---|---|
| `--color-primary` | `#6C5CE7` | 主按钮、我的气泡、选中态、人格卡片高亮 |
| `--color-secondary` | `#00CEC9` | 匹配动效、运势卡片点缀 |
| `--color-accent` | `#FD79A8` | Like 按钮、匹配通知 |
| `--color-bg` | `#0F0F1A` | 页面背景 |
| `--color-surface` | `#1A1A2E` | 卡片、输入框、对方气泡 |
| `--color-error` | `#FF6B6B` | Pass 按钮、举报 |
| `--radius-sm/md/lg/xl` | `8/12/16/24px` | 同上 |
| `--font-heading/body` | `Inter` | 统一西文/中文 |

## 4. 关键交互状态

| 场景 | 状态处理 |
|---|---|
| 加载中 | 骨架屏（卡片轮廓灰色闪烁） |
| 空数据 | 图标 + 说明文案 + CTA 按钮 |
| 网络错误 | 顶部 SnackBar + 重试 |
| WebSocket 断连 | 聊天页顶部灰色条 "连接中..."，自动重连 |
| 发送失败 | 气泡变红 + 点击重发 |
| 匹配成功 | 全屏弹窗（2.5），2s 自动消失，可点击聊天 |
| Token 过期 | 静默刷新失败 → 下次 API 401 → 跳发现页（无需登出） |
| 未注册拦截 | 半屏弹窗（2.2），不跳转页面 |

## 5. 减法清单（相比 Soul 去掉了什么）

| Soul 有 | Spark 去掉 | 原因 |
|---|---|---|
| 星球旋转匹配动效 | 无，直接卡片滑动 | 减法，卡片即核心 |
| 语音速配 | 无 | V1.1 才有语音消息 |
| 群聊派对 | 无 | 不做社区 |
| 朋友圈/瞬间 Feed | 无 | 不做社区 |
| 复杂的虚拟形象拼装 | 6-8 预设 | 减法，形象不是核心 |
| 等级/积分/签到 | 无 | 不做游戏化 |
| 成就/勋章 | 无 | 不做游戏化 |
| 头像边框/特效/装饰 | 无 | 减法 |
| 暖色/卡通/元宇宙风 | 暗色/简约/霓虹 | 差异化视觉 |
